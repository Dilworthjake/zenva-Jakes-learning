# Patient records application

## store and display patient information

- Display first and last names, D.O.B, height, weight, and medication status
- add new patients with the above fields

### functional requirements

- display patient data in a table
- open and close patient intake form
- capture and validate inputs
- add new patient data to the list

## User Stories

- As a medical professional I want a table of all the patients so I can view their information all in one place
- As a medical professional I want to be able to enter patient information so that I can save a new patient to my records

- As a medical professional I want an add patient button so that I can display the patient intake form
- As a medical professional I want input validation so that I can ensure patient information is captured accurately
