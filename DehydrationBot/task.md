# Dehydration bot

## project definition

### To diagnose a user's state of dehydration based on a short questionnaire

- Yes or no type answers
- Previous responses determine next question
- Severe / some/ no dehydration

### To retrieve and add dehydration diagnoses

- Display a list of patients and diagnoses
- Store new diagnoses in the list

## Design

### Main functions

1. Run a new diagnosis and store results
2. Display a list of previous patients and diagnoses

### Style

Text based program in the terminal

1. Display prompt/question
2. process user text input
